SQL Server License 說明

  • SQL Server新版一出舊版就停售，User可以買新版後合法安裝舊版，所有授權計算方式計算以新版為準

  • 目前版本為SQL 2016 (SQL 2017 on linux 2017 年底可能會出)，共分為Standard / Enterprise兩個版本

    • Standard版 : 有Server + CAL(適合小企業) / by Core (適合大部分的單位) 兩種採購模式
    • Enterprise版 : 只有by Core的計價模式

  • by Core的計價模式 : 基本消費額 4 Core，後續 2 core / 2 core加購

  • 若使用實體主機安裝SQL Server，SQL License採購 core 數必須等於實體的Server Core數(不含Hyper-threading模擬出來的)，Microsoft不接受安裝上去只enable N個core這種說法

  • Cluster HA架構

    • 在Active / Passive (Standby)架構下，Passive那台SQL一定不可以拿來做做任何事情(ex. 跑Report)，不然就是要收錢
    • 從SQL 2014版起 (ie, from 2014年底開始)
      • 如果客戶單位有和Microsoft簽Software Assurance的約(就是固定繳年費)，Passive (Standby)那台Server是Free的
      • 如果客戶單位沒有Software Assurance的保護傘，Passive (Standby)那台Server請買與機器相同core數的SQL License
    • 工業局共同供應契約的SQL Server License，並不具備Software Assurance的使用權利
