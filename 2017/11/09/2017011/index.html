原廠的回覆設定 CompatibilityOptions.GrowAutofit 選項為 false 就可以了，如下，

果然設定 Document 的 CompatibilityOptions.GrowAutofit 為 false ， PDF 中的 table 就不會超過頁面了，詳細可以參考DocumentBuilder InsertHtml Save as pdf will exceed page。
